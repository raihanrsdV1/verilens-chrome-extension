Verilens — screens to import into Figma

Use the html.to.design Figma plugin to import each URL below as editable layers. Steps are at the bottom.

① Website — landing page ② Extension popup — Settings ③ Extension popup — Help ④ Website — Pro unlock / success

How to import into Figma

  1. Start the local server from the repo root: python3 -m http.server 8000
  2. In Figma: Menu → Plugins → Manage plugins, search "html.to.design", install it (free).
  3. Open a Figma file → Plugins → html.to.design → Import.
  4. Paste a full URL, e.g. http://localhost:8000/website/preview/popup-settings.html, and import. It drops in as an editable frame with real text/vector layers.
  5. Repeat for each of the 4 URLs above. (Full URLs: /website/index.html, /website/preview/popup-settings.html, /website/preview/popup-help.html, /website/success.html.)

Note: html.to.design needs a reachable URL. If the plugin can't see localhost, use its "paste HTML" mode, or run ngrok http 8000 and import the ngrok URL instead.
